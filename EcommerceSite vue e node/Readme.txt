SiteEcommerce

Descrição

O projeto SiteEcommerce tem como objetivo oferecer uma solução completa para comerciantes que precisam de uma loja online. Ele permite criar, gerenciar e manter uma plataforma de e-commerce.

Tecnologias Utilizadas

Vue.js

HTML5

CSS

Node.js

JavaScript

TypeScript

Instalação

Para rodar o projeto localmente, siga os passos abaixo:

Clone o repositório:

git clone https://github.com/VitorIkeziri/projects/tree/main/EcommerceSite%20vue%20e%20node/SiteEcommerce

Acesse o diretório do projeto:

cd siteecommerce

Instale as dependências do frontend:

cd frontend
npm install

Instale as dependências do backend:

cd ../backend
npm install

Inicie o servidor backend:

node server.js

Inicie o frontend:

npm run dev

Uso

Após a instalação, acesse o projeto no navegador.

Estrutura de Pastas

frontend/: Contém o código do cliente desenvolvido com Vue.js

backend/: Contém a API e o servidor construídos com Node.js
