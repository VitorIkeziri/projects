- Projeto: Análise de Vendas com Dashboard (Python + Jupyter)

Este projeto demonstra habilidades em análise de dados com Python, desde a leitura e limpeza de dados até a geração de insights visuais com gráficos interativos.


 - Objetivo

Realizar a análise de um conjunto de dados de vendas fictício, com o intuito de:

    Identificar os produtos mais vendidos

    Calcular o faturamento mensal

    Analisar categorias de produtos

    Observar tendências de vendas ao longo do tempo

 - Estrutura do Projeto

    vendas.xlsx: Base de dados com 3 abas:

        vendas: informações das vendas (data, produto, cliente, quantidade, preço)

        produtos: dados sobre os produtos (nome, categoria)

        clientes: dados sobre os clientes (nome, região)

    analise_vendas.ipynb: Notebook Jupyter com toda a análise e visualizações.
    
 - Ferramentas Utilizadas

    Python 3.x

    Jupyter Notebook

    Bibliotecas:

        pandas

        matplotlib

        seaborn

        plotly

 - Insights Gerados

    Produto mais vendido (quantidade)

    Produto com maior faturamento

    Faturamento por categoria

    Faturamento mensal

    Tendência de vendas ao longo do tempo (gráfico interativo)
